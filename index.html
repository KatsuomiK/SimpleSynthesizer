Web MIDI Synthesizer (WIP)

Web MIDI API and Web Audio API experiment. Chrome 43 or later is required.
The features are very limited for now. Currently you can:

- play with MIDI keyboard
- play with PC keys (C key = C4, Space = Damper pedal)
- play test SMF
- play simple MML
