Web MIDI Synthesizer

Web MIDI API and Web Audio API simple keyboard. You can play with MIDI keys or PC keys.
(C key = C4, Space = Damper Pedal)
Chrome 43 or later is required.

Programmed by @KatsuomiK / @k0rin (in Japanese)


		
MIDI Implementation Chart

		

			
MIDI Input
			

			
Ch. 1 Note On/Off (without velocity)

			Ch. 1 Pitch Bend

			Ch. 1 CC 1 (Modulation wheel)

			Ch. 1 CC 64 Damper Pedal On/Off
