Web MIDI Synthesizer (WIP)

Web MIDI API and Web Audio API experiment works on Chrome and iOS Safari.
Currently you can:

- play with MIDI keyboard
- play with PC keys (C key = C4, Space = Damper pedal)
- play test SMF
- play simple MML
