# Web MIDI Synthesizer

Web MIDI API and Web Audio API simple keyboard.

Working demo is at http://framesynthesis.com/experiments/WebMIDISynthesizer/

# Build

$ npm install  
$ npm run watch

# MIDI Implementation Chart

## MIDI Input

- Ch. 1 Note On/Off (witout velocity)
- Ch. 1 Pitch Bend

That's all for now :)
